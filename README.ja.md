# freq-analyzer

WebAudioAPIを使用したFrequency Analyzerです。

## デモ
https://code4fukui.github.io/freq-analyzer/

## 機能
- リアルタイムオーディオ解析
- 周波数バンドの表示
- 最大値表示
- グラフ表示

## 使い方
ブラウザで開くと、マイクからの入力音声のスペクトルが表示されます。

「enable show value on the table」のチェックボックスでテーブルの表示/非表示を切り替えられます。

「clear max value」ボタンで最大値表示をリセットできます。
