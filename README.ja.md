# freq-analyzer

WebAudio APIとChart.jsを使用して構築された、シンプルなリアルタイムの音声周波数アナライザーです。

## デモ

https://code4fukui.github.io/freq-analyzer/

このアプリケーションはマイク入力をキャプチャし、さまざまな音声周波数ビンの強度を表すリアルタイムの折れ線グラフ（マルチラインチャート）を表示します。グラフの下には、各周波数の現在値とピーク値を示す詳細なテーブルが表示されます。

## 特徴

- **リアルタイム解析**: WebAudio APIを使用してマイクの音声をキャプチャし、リアルタイムで周波数解析を行います。
- **時系列チャート**: 周波数スペクトルを時系列の折れ線グラフとして可視化し、各周波数ビンの最新値を表示します。
- **データテーブル**: 各周波数ビンの生のFFT値と、最後のリセット以降に記録された最大値を並べて表示します。
- **シンプルな操作**: データテーブルの表示/非表示を切り替えるチェックボックスと、記録された最大値をクリアするボタンを備えています。

## 使い方

1. モダンブラウザで[デモページ](https://code4fukui.github.io/freq-analyzer/)を開きます。
2. マイクへのアクセス許可を求められたら、許可します。
3. グラフにリアルタイムの周波数スペクトルが即座に表示され始めます。
4. データテーブルはデフォルトで表示されます。「enable show value on the table」のチェックを外すと非表示になります。
5. 「clear max value」ボタンをクリックすると、テーブルに記録されているピーク値がリセットされます。

## 技術的詳細

- **API**: WebAudio APIの`AnalyserNode`を使用して、音声ストリームに対して高速フーリエ変換（FFT）を実行します。
- **設定**:
  - `fftSize`: 32（結果として16の周波数ビンを生成）
  - `smoothingTimeConstant`: 0
- **可視化**: グラフの描画には[Chart.js](https://code4fukui.github.io/Chart-es/Chart.js)を使用しています。
