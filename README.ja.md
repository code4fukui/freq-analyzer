# freq-analyzer

WebAudioAPIを使用したリアルタイムのオーディオ周波数解析ツールです。アプリケーションを起動するとマイクへのアクセスを求めるため、許可することで周波数スペクトルをグラフとして表示できます。周波数値のテーブル表示も可能です。

## デモ
https://code4fukui.github.io/freq-analyzer/

## 機能
- リアルタイムのオーディオ周波数解析
- 周波数スペクトルのグラフ表示
- 周波数値のテーブル表示（オプション）
- 最大値の表示とリセット

## 使い方
1. ブラウザで Web ページを開く
2. マイクへのアクセスを許可する
3. 周波数スペクトルがグラフに表示される
4. 「clear max value」ボタンで最大値表示をリセットできる
5. 「enable show value on the table」のチェックボックスでテーブル表示の切り替えが可能
